## 近期优化 🔧

### v1.1.0（进行中）

#### ✅ 已完成
- [x] 提升代码上下文检测准确度
  - 改进 `isCodeContext()` 从简单字符串匹配到正则匹配
  - 改进 `hasComplexitySignals()` 从简单字符串匹配到正则匹配
  - 提高"写脚本"与"系统架构"的区分能力
  - 优化代码块检测逻辑
  
#### 📝 待办
- [ ] **Fallback 模型自动切换**
  - 当 primary 模型调用失败时，自动切换到 fallback 继续任务
  - 需要研究 OpenClaw 的事件 hook 系统
  - 需要监听模型调用失败事件
  - 优先级：中
- [ ] 增加更多测试用例
  - 边界情况测试
  - 真实用户消息样本
  - 性能基准测试
- [ ] 提升关键词库覆盖率
  - 补充英文技术术语
  - 新兴领域关键词（Web3、区块链等）
- [ ] 优化 Sigmoid 校准参数
  - 根据真实数据调整中点和 k 斜率
